# Reverse Integer

## Условие

Дано 32-битное целое число `x`. Необходимо разворошить цифры этого числа и вернуть результат.

Если развёрнутое число выходит за границы диапазона 32-битного целого числа **[-2^31, 2^31 - 1]**, вернуть 0.

Примечание: предполагается, что среда не позволяет сохранять 64-битные целые числа.

## Примеры

Input: `x = 123`  
Output: `321`

Input: `x = -123`  
Output: `-321`

Input: `x = 120`  
Output: `21`

Input: `x = 0`  
Output: `0`

Input: `x = 1534236469`  
Output: `0` (развёрнутое число выходит за границы диапазона)

## Подход

- Проверяем знак числа и запоминаем его
- Работаем с абсолютным значением числа
- Извлекаем цифры справа налево (используя операцию `% 10`)
- Строим развёрнутое число, умножая результат на 10 и добавляя новую цифру
- **Перед добавлением каждой цифры** проверяем, не превысит ли это границу INT_MAX
- Применяем сохранённый знак к результату

## Сложность

- Time: O(log n) — где n — количество цифр в числе (примерно log₁₀ от x)
- Space: O(1) — используем только несколько переменных

## Идея

Основная идея — извлечение цифр числа с помощью арифметических операций без преобразования в строку. Проверка переполнения должна быть выполнена **до** добавления цифры, чтобы избежать переполнения при самой проверке.
