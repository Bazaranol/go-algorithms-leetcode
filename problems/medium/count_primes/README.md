# Count Primes

## Условие

Дано целое число `n`. Необходимо вернуть количество простых чисел, которые **меньше** `n`.

**Примечание:** простое число — это натуральное число, которое делится только на 1 и на себя.

## Примеры

Input: `n = 10`  
Output: `4`  
Объяснение: Простые числа меньше 10 это `[2, 3, 5, 7]`. Всего 4 числа.

Input: `n = 15`  
Output: `6`  
Объяснение: Простые числа меньше 15 это `[2, 3, 5, 7, 11, 13]`. Всего 6 чисел.

Input: `n = 2`  
Output: `0`  
Объяснение: Нет простых чисел меньше 2.

## Подход

Используется классический алгоритм **Решето Эратосфена** (Sieve of Eratosthenes):

1. Создаём булев массив размером `n`, инициализируем все значения в `true`
2. Отмечаем 0 и 1 как не простые (`false`)
3. Проходим по всем числам `i` от 2 до √n:
   - Если `i` отмечено как простое, то отмечаем все его кратные (i², i²+i, i²+2i, ...) как не простые
4. Подсчитываем оставшиеся числа, отмеченные как `true`

## Сложность

- Time: O(n log log n) — оптимальная сложность для поиска всех простых чисел до n
- Space: O(n) — булев массив размером n

## Идея

Решето Эратосфена — это один из самых эффективных алгоритмов для нахождения всех простых чисел до определённого числа. Ключевая оптимизация — мы начинаем отмечать кратные с числа `i²`, а не с `2i`, потому что все кратные `i` меньше `i²` уже были отмечены более меньшими числами.
