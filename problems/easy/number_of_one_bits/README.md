# Number of 1 Bits

## Условие

Напишите функцию, которая подсчитывает количество установленных битов (единиц, 1s) в двоичном представлении положительного целого числа.

Эта задача также известна как **Hamming Weight** или **Population Count**.

## Примеры

Input: `n = 11`  
Output: `3`  
Объяснение: 11 в двоичной системе = 1011 (три единицы)

Input: `n = 128`  
Output: `1`  
Объяснение: 128 в двоичной системе = 10000000 (одна единица)

Input: `n = 2147483647`  
Output: `31`  
Объяснение: 2147483647 = 01111111111111111111111111111111 (31 единица)

## Подход

Используется остроумный битовый трюк: операция `n & (n - 1)`

**Как это работает:**
- Выражение `n & (n - 1)` удаляет самый правый установленный бит из числа n
- Цикл продолжается до тех пор, пока n не станет 0
- Количество итераций = количество единиц в двоичном представлении

**Пример для n = 11 (1011₂):**
1. n = 11 (1011₂), n-1 = 10 (1010₂), n & (n-1) = 1010₂ = 10, count = 1
2. n = 10 (1010₂), n-1 = 9 (1001₂), n & (n-1) = 1000₂ = 8, count = 2
3. n = 8 (1000₂), n-1 = 7 (0111₂), n & (n-1) = 0000₂ = 0, count = 3
4. n = 0, цикл завершён, результат = 3 ✓

## Сложность

- Time: O(k) — где k количество установленных битов (максимум 32 для int)
- Space: O(1) — используем только несколько переменных

## Идея

Ключевая идея — операция `n & (n - 1)`. Когда мы вычитаем 1 из n:
- Все нули справа от самого правого установленного бита становятся единицами
- Самый правый установленный бит становится нулём

Битовое И (AND) обнуляет все эти биты, оставляя только старшие биты нетронутыми. Поэтому каждая итерация удаляет ровно один установленный бит, и алгоритм завершается за O(количество единиц) операций, а не за O(количество битов).

## Альтернативный подход

Можно просто сдвигать биты вправо и проверять последний бит:
```go
func NumberOfOneBits(n int) int {
    count := 0
    for n > 0 {
        if n & 1 == 1 {
            count++
        }
        n >>= 1
    }
    return count
}
```
Но это требует O(32) операций (количество битов), тогда как мое решение требует O(k) где k — количество единиц.
