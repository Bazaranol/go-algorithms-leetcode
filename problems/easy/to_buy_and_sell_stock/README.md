# Best Time to Buy and Sell Stock

## Условие

Дан массив целых чисел `prices`, где `prices[i]` — цена акции в день `i`. 

Найти максимальную прибыль, которую можно получить от покупки и продажи акции один раз. Если получить прибыль невозможно, вернуть 0.

Примечание: вы должны продать акцию **после** её покупки.

## Пример

Input: `[7, 1, 5, 3, 6, 4]`  
Output: `5`  
Объяснение: Купить при цене 1, продать при цене 6, прибыль = 6 - 1 = 5.

Input: `[7, 6, 4, 3, 1]`  
Output: `0`  
Объяснение: Нет возможности получить прибыль, цены только падают.

## Подход

- Проходим по массиву слева направо
- Отслеживаем минимальную цену, которую видели до сих пор
- Для каждой цены вычисляем потенциальную прибыль (текущая цена - минимальная цена)
- Обновляем максимальную прибыль и минимальную цену

## Сложность

- Time: O(n) — один проход по массиву
- Space: O(1) — используем только константное количество переменных

## Идея

Ключевая идея — если мы знаем минимальную цену до текущей позиции, то можно быстро вычислить максимальную прибыль в этот момент. Нет необходимости проверять все пары покупки-продажи.
