# PalindromeNumber

## Условие

 Дано число, нужно вернуть true, если оно является палиндромом, либо false в противном случае.

---

## Примеры

Input: 100 
Output: false 

Input: 101
Output: true 

---

## Подход

- Запоминаем оригинал и создаем реверс переменную
- Реверс переменную заполняем с правого края оригинала, записываем в начало

---

## Сложность

* Time: ?
* Space: O(n)

---

## Идея

- Идем по циклу пока дубликат больше 0
- Делим с остатком, остаток прибавлем к реверсу
