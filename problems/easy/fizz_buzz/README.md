# Fizz Buzz

## Условие

Дано целое число `n`. Необходимо вернуть массив строк длины `n`, где для каждого числа от 1 до n:

- Если число делится на 3, добавить `"Fizz"`
- Если число делится на 5, добавить `"Buzz"`
- Если число делится и на 3, и на 5, добавить `"FizzBuzz"`
- Иначе добавить строковое представление числа

## Примеры

Input: `n = 10`  
Output: `["1", "2", "Fizz", "4", "Buzz", "Fizz", "7", "8", "Fizz", "Buzz"]`

Input: `n = 15`  
Output: `["1", "2", "Fizz", "4", "Buzz", "Fizz", "7", "8", "Fizz", "Buzz", "11", "Fizz", "13", "14", "FizzBuzz"]`

Input: `n = 3`  
Output: `["1", "2", "Fizz"]`

## Подход

- Создаём срез строк размером `n`
- Проходим по каждому индексу от 0 до n-1
- Вычисляем актуальное число как `index + 1`
- Проверяем условия в порядке приоритета:
  1. Делится ли и на 3, и на 5 → добавить `"FizzBuzz"`
  2. Делится ли на 3 → добавить `"Fizz"`
  3. Делится ли на 5 → добавить `"Buzz"`
  4. Иначе → преобразовать число в строку

## Сложность

- Time: O(n) — один проход по всем числам от 1 до n
- Space: O(n) — результирующий массив размером n

## Идея

Главное — правильный порядок проверки условий. Нужно сначала проверить делится ли число на оба 3 и 5, иначе это число никогда не получит `"FizzBuzz"`. Проверка на `3 && 5` перед отдельными проверками гарантирует корректный результат.
