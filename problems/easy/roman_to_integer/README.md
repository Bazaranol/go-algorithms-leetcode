# Roman to Integer

## Условие

Дана строка с римским числом, необходимо преобразовать её в integer.

## Пример

Input: "MCMXCIV"  
Output: 1994  

## Подход

- Идём справа налево
- Если текущее значение меньше предыдущего → вычитаем
- Иначе → прибавляем

## Сложность

- Time: O(n)
- Space: O(1)
