# Power of Four

## Условие

Дано целое число `n`. Определить, является ли `n` степенью числа 4.

Целое число `n` является степенью 4, если существует целое число `x` такое, что `n == 4^x`.

## Примеры

Input: `n = 16`  
Output: `true`  
Объяснение: 4^2 = 16

Input: `n = 5`  
Output: `false`

Input: `n = 1`  
Output: `true`  
Объяснение: 4^0 = 1

Input: `n = 8`  
Output: `false`  
Объяснение: 8 это 2^3, степень 2, но не степень 4

---

## Два подхода

В этом решении реализованы **два способа**:

### 1. PowerOfFour (Деление в цикле)

Делим число на 4 в цикле, пока оно делится нацело:
- Если в конце получилось 1 → число степень 4
- Если в какой-то момент остаток не равен 0 → не степень 4

**Сложность:** Time O(log₄ n), Space O(1)

### 2. PowerOfFourByBit (Битовая операция с маской)

Используем три условия одновременно:

```go
n > 0 && (n&(n-1)) == 0 && (n&0x55555555) != 0
```

**Разберемся:**

1. **`n > 0`** — число положительное

2. **`n&(n-1) == 0`** — число является степенью 2
   - Степень 4 это всегда степень 2 (4 = 2²)
   - Примеры: 4 (100₂), 16 (10000₂), 64 (1000000₂)
   - Так что сначала проверяем, что это степень 2

3. **`n&0x55555555 != 0`** — бит установлен в правильной позиции
   - 0x55555555 = 01010101010101010101010101010101₂
   - Степени 4: 1 (1₂), 4 (100₂), 16 (10000₂), 64 (1000000₂), 256 (100000000₂)...
   - Степени 2, но не 4: 2 (10₂), 8 (1000₂), 32 (100000₂)...
   - Маска 0x55555555 отмечает позиции, где может быть бит для степеней 4
   - Позиции: 0, 2, 4, 6, 8... (чётные позиции)

**Примеры:**

| n | Двоичная | Степень 2? | Чётная позиция? | Результат |
|---|----------|-----------|-----------------|-----------|
| 1 | 1 | ✓ (1 = 2⁰) | ✓ (позиция 0) | ✓ |
| 4 | 100 | ✓ (4 = 2²) | ✓ (позиция 2) | ✓ |
| 16 | 10000 | ✓ (16 = 2⁴) | ✓ (позиция 4) | ✓ |
| 2 | 10 | ✓ (2 = 2¹) | ✗ (позиция 1) | ✗ |
| 8 | 1000 | ✓ (8 = 2³) | ✗ (позиция 3) | ✗ |

**Сложность:** Time O(1), Space O(1) — три условия, никаких циклов!

---

## Сравнение подходов

| Решение | Время | Память | Примечание |
|---|---|---|---|
| Деление в цикле | O(log₄ n) | O(1) | Понятный, прямолинейный |
| Битовая операция | O(1) | O(1) | Очень быстро, но требует понимания битов |

## Идея

Ключевая идея второго подхода — степени 4 это подмножество степеней 2. Степень 2 может быть на любой позиции, но степень 4 может быть только на **чётных** позициях (0, 2, 4, 6...). Маска 0x55555555 кодирует эти чётные позиции.
