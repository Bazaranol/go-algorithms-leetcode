# Power of Two

## Условие

Дано целое число `n`. Определить, является ли `n` степенью числа 2.

Целое число `n` является степенью 2, если существует целое число `x` такое, что `n == 2^x`.

## Примеры

Input: `n = 1`  
Output: `true`  
Объяснение: 2^0 = 1

Input: `n = 16`  
Output: `true`  
Объяснение: 2^4 = 16

Input: `n = 3`  
Output: `false`

Input: `n = 4`  
Output: `true`  
Объяснение: 2^2 = 4

Input: `n = 5`  
Output: `false`

---

## Два подхода

В этом решении реализованы **два способа**:

### 1. PowerOfTwo (Деление в цикле)

Делим число на 2 в цикле, пока оно делится нацело:
- Если в конце получилось 1 → число степень 2
- Если в какой-то момент остаток не равен 0 → не степень 2

**Сложность:** Time O(log n), Space O(1)

### 2. PowerOfTwoByBit (Битовая операция)

Используем битовую операцию: `n & (n-1) == 0`

**Почему это работает:**
- Степень 2 в двоичной системе имеет ровно один бит, установленный в 1: например, 8 = 1000₂, 16 = 10000₂
- Число (n-1) будет иметь все биты 1 там, где n был 0, и наоборот
- Битовое И (`&`) число n и (n-1) даст 0 только для степеней 2

Примеры:
- 8 (1000₂) & 7 (0111₂) = 0 ✓
- 5 (0101₂) & 4 (0100₂) = 4 ≠ 0 ✗

**Сложность:** Time O(1), Space O(1) — это самый быстрый способ!

---

## Сравнение подходов

| Решение | Время | Память | Примечание |
|---|---|---|---|
| Деление в цикле | O(log n) | O(1) | Интуитивно понятно |
| Битовая операция | O(1) | O(1) | Наиболее эффективно |

## Идея

Битовый способ — это классический трюк для проверки степеней 2. Степени 2 имеют уникальное двоичное представление (один установленный бит), что позволяет проверить это за одну операцию.
